# flow-full-pipeline

**Пресет: объёмная задача, полный контур** — пошаговая цепочка с изоляцией контекста по фазам. Используйте, когда нужно **явно** пройти: аналитика → (при необходимости) разведку и архитектурные шаги → реализацию → ревью → приёмку.

**Обязательные ссылки пакета:**

- Маршрут и масштаб: **`pauk/routing/PROTOCOL.md`**
- Пресет и вариации: **`pauk/subagents/WORKFLOW.md`**, раздел **«Предопределённые сценарии»**
- Шаблон плана: **`pauk/routing/ARTIFACT-ROUTER.md`**
- Каталог навыков: **`pauk/routing/SKILLS-CATALOG.md`**
- Политика: **`pauk/policy/A-INVARIANTS.md`**

**Фиксация в `route`:** `scale: full`. Первое сообщение после вызова команды — **короткое** по `PROTOCOL` (при необходимости), затем шаг 2: заполнение **`ARTIFACT-ROUTER.md` ролью `task-analyst`**, согласование с человеком, далее исполнение фаз **по таблице в пресете** с пересборкой `route.skills` при смене предмета задачи.

**Не** подставляйте пути к платформе, ИБ и секреты: только регламент репозитория (`pauk/reference/infobase-sync.md` и файлы проекта, если применимо).
