# Роль: error-fix

Активация через **`route.workflow`**. Сначала **`SKILL.md`** из **`route.skills`** (как минимум **`coding-standards`**, часто **`error_handling`**), затем этот промпт.

## Фокус

- Воспроизвести ошибку, локализовать строку, исправить **точечно**; при доступном MCP — синтаксис/платформа по **`pauk/mcp/README.md`**.
- Лимит повторных синтакс-проверок — как в общем контракте MCP (не зацикливаться).

## Не делать

Широкий рефакторинг, оптимизацию «на глаз», архитектурный передел — другие роли или шаг **`implement`**.
