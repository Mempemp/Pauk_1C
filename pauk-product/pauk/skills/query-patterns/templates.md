# Шаблоны запросов (копировать и адаптировать)

## Шаблон 1: один запрос вместо цикла

```bsl
// Получаем ВСЕ данные одним запросом
Запрос = Новый Запрос;
Запрос.Текст =
"ВЫБРАТЬ
|   Номенклатура.Ссылка КАК Ссылка,
|   Номенклатура.Наименование КАК Наименование,
|   Номенклатура.Цена КАК Цена
|ИЗ
|   Справочник.Номенклатура КАК Номенклатура
|ГДЕ
|   Номенклатура.Ссылка В (&МассивСсылок)";

Запрос.УстановитьПараметр("МассивСсылок", МассивНоменклатуры);
Результат = Запрос.Выполнить();

// Создаём соответствие для быстрого доступа
СоответствиеДанных = Новый Соответствие;
Выборка = Результат.Выбрать();
Пока Выборка.Следующий() Цикл
    СоответствиеДанных.Вставить(Выборка.Ссылка,
        Новый Структура("Наименование, Цена", Выборка.Наименование, Выборка.Цена));
КонецЦикла;

// Используем в цикле БЕЗ запросов
Для Каждого Строка Из Документ.Товары Цикл
    Данные = СоответствиеДанных.Получить(Строка.Номенклатура);
    Если Данные <> Неопределено Тогда
        Строка.Цена = Данные.Цена;
    КонецЕсли;
КонецЦикла;
```

## Шаблон 2: пакетный запрос с временными таблицами

```bsl
Запрос = Новый Запрос;
Запрос.Текст =
"// Этап 1: Отбираем документы
|ВЫБРАТЬ
|   Документ.Ссылка КАК Ссылка,
|   Документ.Контрагент КАК Контрагент,
|   Документ.Сумма КАК Сумма
|ПОМЕСТИТЬ втДокументы
|ИЗ
|   Документ.Реализация КАК Документ
|ГДЕ
|   Документ.Дата МЕЖДУ &ДатаНачала И &ДатаОкончания
|   И Документ.Проведен
|
|ИНДЕКСИРОВАТЬ ПО
|   Контрагент
|;
|
|////////////////////////////////////////////////////////////////////////////////
|// Этап 2: Группируем по контрагентам
|ВЫБРАТЬ
|   втДокументы.Контрагент КАК Контрагент,
|   СУММА(втДокументы.Сумма) КАК ОбщаяСумма,
|   КОЛИЧЕСТВО(РАЗЛИЧНЫЕ втДокументы.Ссылка) КАК КоличествоДокументов
|ПОМЕСТИТЬ втИтоги
|ИЗ
|   втДокументы
|СГРУППИРОВАТЬ ПО
|   втДокументы.Контрагент
|;
|
|////////////////////////////////////////////////////////////////////////////////
|// Этап 3: Соединяем с остатками
|ВЫБРАТЬ
|   втИтоги.Контрагент,
|   втИтоги.ОбщаяСумма,
|   втИтоги.КоличествоДокументов,
|   ЕСТЬNULL(Остатки.СуммаОстаток, 0) КАК Задолженность
|ИЗ
|   втИтоги
|       ЛЕВОЕ СОЕДИНЕНИЕ РегистрНакопления.Взаиморасчеты.Остатки КАК Остатки
|       ПО втИтоги.Контрагент = Остатки.Контрагент";

Запрос.УстановитьПараметр("ДатаНачала", ДатаНачала);
Запрос.УстановитьПараметр("ДатаОкончания", ДатаОкончания);
Результат = Запрос.Выполнить();
```

Имена документов и регистров заменить на метаданные своей конфигурации.

## Шаблон 3: виртуальная таблица — параметры внутри

```bsl
Запрос.Текст =
"ВЫБРАТЬ
|   Остатки.Номенклатура,
|   Остатки.Склад,
|   Остатки.КоличествоОстаток
|ИЗ
|   РегистрНакопления.ТоварыНаСкладах.Остатки(
|       &ДатаОстатков,
|       Номенклатура В (&СписокНоменклатуры)
|           И Склад = &Склад
|   ) КАК Остатки";

Запрос.УстановитьПараметр("ДатаОстатков", ДатаОстатков);
Запрос.УстановитьПараметр("СписокНоменклатуры", МассивНоменклатуры);
Запрос.УстановитьПараметр("Склад", СкладСсылка);
```

## Шаблон 4: пакет через ТаблицуЗначений

```bsl
// Подготавливаем таблицу значений
ТаблицаДанных = Новый ТаблицаЗначений;
ТаблицаДанных.Колонки.Добавить("Штрихкод", Новый ОписаниеТипов("Строка", , Новый КвалификаторыСтроки(13)));
ТаблицаДанных.Колонки.Добавить("Количество", Новый ОписаниеТипов("Число", Новый КвалификаторыЧисла(15, 3)));

Для Каждого Строка Из ДанныеИмпорта Цикл
    НоваяСтрока = ТаблицаДанных.Добавить();
    НоваяСтрока.Штрихкод = Строка.Штрихкод;
    НоваяСтрока.Количество = Строка.Количество;
КонецЦикла;

// Один запрос обрабатывает все данные
Запрос = Новый Запрос;
Запрос.УстановитьПараметр("ДанныеИмпорта", ТаблицаДанных);
Запрос.Текст =
"ВЫБРАТЬ
|   Данные.Штрихкод,
|   Данные.Количество
|ПОМЕСТИТЬ втДанные
|ИЗ
|   &ДанныеИмпорта КАК Данные
|;
|
|////////////////////////////////////////////////////////////////////////////////
|ВЫБРАТЬ
|   Штрихкоды.Номенклатура,
|   втДанные.Количество
|ИЗ
|   втДанные
|       ВНУТРЕННЕЕ СОЕДИНЕНИЕ РегистрСведений.Штрихкоды КАК Штрихкоды
|       ПО втДанные.Штрихкод = Штрихкоды.Штрихкод";

Результат = Запрос.Выполнить();
```
