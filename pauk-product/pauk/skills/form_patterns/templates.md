# Шаблоны: модуль управляемой формы

Индекс правил и приоритеты — в [SKILL.md](SKILL.md). Имена модулей (`ОбщегоНазначения`, `РаботаСКурсамиВалют` и т.д.) — из иллюстраций для типовых конфигураций; подставьте вызовы вашей библиотеки.

---

## Шаблон 1: все данные за один серверный вызов (не дергать сервер по три раза подряд)

```bsl
// КЛИЕНТ: один вызов для всех данных
&НаКлиенте
Процедура КонтрагентПриИзменении(Элемент)
    ДанныеЗаполнения = ПолучитьВсеДанныеОдинВызов(Объект.Контрагент, Объект.Дата);

    Объект.Договор = ДанныеЗаполнения.Договор;
    Объект.Валюта = ДанныеЗаполнения.Валюта;
    КурсВалюты = ДанныеЗаполнения.Курс;
КонецПроцедуры

// СЕРВЕР БЕЗ КОНТЕКСТА: все данные за один раз
&НаСервереБезКонтекста
Функция ПолучитьВсеДанныеОдинВызов(КонтрагентСсылка, ДатаДокумента)
    Результат = Новый Структура("Договор, Валюта, Курс");

    РеквизитыКонтрагента = ОбщегоНазначения.ЗначенияРеквизитовОбъекта(
        КонтрагентСсылка, "ДоговорПоУмолчанию");

    Результат.Договор = РеквизитыКонтрагента.ДоговорПоУмолчанию;

    Если ЗначениеЗаполнено(Результат.Договор) Тогда
        ВалютаДоговора = ОбщегоНазначения.ЗначениеРеквизитаОбъекта(
            Результат.Договор, "ВалютаВзаиморасчетов");
        Результат.Валюта = ВалютаДоговора;
        Результат.Курс = РаботаСКурсамиВалют.ПолучитьКурсВалюты(ВалютаДоговора, ДатаДокумента);
    КонецЕсли;

    Возврат Результат;
КонецФункции
```

## Шаблон 2: асинхронный диалог с оповещением

```bsl
// Команда с подтверждением
&НаКлиенте
Процедура УдалитьСтроку(Команда)
    ПоказатьВопрос(
        Новый ОписаниеОповещения("ПослеПодтверждения", ЭтотОбъект),
        НСтр("ru = 'Удалить выбранную строку?'"),
        РежимДиалогаВопрос.ДаНет);
КонецПроцедуры

// Callback после ответа
&НаКлиенте
Процедура ПослеПодтверждения(Результат, ДополнительныеПараметры) Экспорт
    Если Результат = КодВозвратаДиалога.Да Тогда
        Объект.Товары.Удалить(Элементы.Товары.ТекущаяСтрока);
    КонецЕсли;
КонецПроцедуры
```

## Шаблон 3: реквизит формы в значение и обратно

```bsl
&НаСервере
Процедура ВыполнитьДействиеНаСервере()
    ДокументОбъект = РеквизитФормыВЗначение("Объект");

    ДокументОбъект.ЗаполнитьПоУмолчанию();

    ЗначениеВРеквизитФормы(ДокументОбъект, "Объект");
КонецПроцедуры
```

Для преобразования произвольных данных формы (не только основного реквизита «Объект») используйте **`ДанныеФормыВЗначение` / `ЗначениеВДанныеФормы`** по документации платформы для соответствующего типа данных формы.

## Шаблон 4: открытие формы с параметрами и callback

```bsl
&НаКлиенте
Процедура ОткрытьФормуПодбора(Команда)
    ПараметрыФормы = Новый Структура;
    ПараметрыФормы.Вставить("Организация", Объект.Организация);
    ПараметрыФормы.Вставить("МножественныйВыбор", Истина);

    ОткрытьФорму("Обработка.Подбор.Форма",
        ПараметрыФормы,
        ЭтотОбъект,
        ,,,
        Новый ОписаниеОповещения("ПослеПодбора", ЭтотОбъект));
КонецПроцедуры

&НаКлиенте
Процедура ПослеПодбора(Результат, ДополнительныеПараметры) Экспорт
    Если Результат = Неопределено Тогда
        Возврат;
    КонецЕсли;

    Для Каждого Элемент Из Результат Цикл
        НоваяСтрока = Объект.Товары.Добавить();
        ЗаполнитьЗначенияСвойств(НоваяСтрока, Элемент);
    КонецЦикла;
КонецПроцедуры
```
