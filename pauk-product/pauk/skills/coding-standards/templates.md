### Шаблон 4: Конкатенация строк
```bsl
// ✅ ПРАВИЛЬНО: массив + СтрСоединить()
ЧастиСтроки = Новый Массив;
Для Каждого Элемент Из Коллекция Цикл
    ЧастиСтроки.Добавить(Элемент.Наименование);
КонецЦикла;
РезультатСтрока = СтрСоединить(ЧастиСтроки, ", ");

// ❌ ПЛОХО: конкатенация в цикле
Результат = "";
Для Каждого Элемент Из Коллекция Цикл
    Результат = Результат + Элемент.Наименование + ", ";
КонецЦикла;
```

### Шаблон 5: Локализация строк
```bsl
// ✅ ПРАВИЛЬНО: НСтр() для пользовательских сообщений
ТекстСообщения = СтрШаблон(
    НСтр("ru = 'Документ %1 от %2 на сумму %3 руб.'"),
    НомерДокумента,
    Формат(ДатаДокумента, "ДЛФ=D"),
    Формат(Сумма, "ЧДЦ=2"));

// ❌ ПЛОХО: строка без НСтр()
ТекстСообщения = "Документ не может быть проведён";
```
