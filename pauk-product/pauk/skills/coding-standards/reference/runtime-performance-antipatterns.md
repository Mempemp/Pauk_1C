# Антипаттерны времени выполнения (циклы, клиент-сервер, кэш)

Индекс правил BSL — в `../SKILL.md`. Здесь — типичные **ошибки производительности** вне текста запроса (для запросов см. навык `query-patterns`). Директивы форм и `&НаСервереБезКонтекста` — подробнее в [directives-forms-cs.md](directives-forms-cs.md).

---

## 1. Множественные вызовы с клиента на сервер подряд

**Проблема:** каждый вызов серверной процедуры с клиента несёт накладные расходы сериализации и сети. Цепочка из нескольких вызовов в одном обработчике без необходимости хуже одного вызова, возвращающего структуру или таблицу с нужными полями.

**Лучше:** одна серверная функция собирает данные в `Структуру` / `ТаблицуЗначений` / соответствие ключей клиенту.

---

## 2. `&НаСервере`, когда достаточно `&НаСервереБезКонтекста`

**Проблема:** лишняя передача контекста формы на сервер, если методу не нужны реквизиты и элементы формы.

**Лучше:** вынести чистую логику в `&НаСервереБезКонтекста` и передать **значения** параметрами (см. `directives-forms-cs.md`).

---

## 3. Повтор одной и той же тяжёлой операции в цикле

**Проблема:** одинаковый расчёт или обращение к данным с теми же ключами (валюта+дата, ссылка+период и т.д.) в каждой итерации цикла.

**Лучше:** кэш в `Соответствие` по составному ключу: сначала `Получить`, при `Неопределено` — вычислить и `Вставить`.

---

## 4. Поиск пары строк вложенными циклами O(n²)

**Проблема:** два вложенных `Для Каждого` для сопоставления по ключу на больших таблицах.

**Лучше:** проиндексировать меньшую коллекцию в `Соответствие` по ключу, затем один проход по второй коллекции с `Получить`.

---

## 5. Глубокая вложенность условий

**Проблема:** много уровней `Если` вложенно — сложно сопровождать и ошибиться в ветках.

**Лучше:** ранние `Возврат` / инвертированные условия (см. `architecture-and-platform.md`, приём «ранний выход») и вынос сложных условий в функции.

---

## 6. Чтение нескольких реквизитов по ссылке через точку в цикле

**Проблема:** цепочки `Объект.Реквизит1`, `Объект.Реквизит2` по одной ссылке в цикле по множеству ссылок дают лишние обращения к данным.

**Лучше:** в конфигурациях на БСП — пакетные методы `ОбщегоНазначения` (см. навык `ssl_patterns` → `reference/common-modules-strings-cs.md`); иначе — один запрос или серверный метод, возвращающий таблицу/соответствие.
