# Шаблоны: транзакции, исключения, блокировки

Индекс правил — в [SKILL.md](SKILL.md). Имена метаданных (`Документ.Название`, `РегистрНакопления.Название`) замените на объекты вашей конфигурации.

---

## Шаблон 1: каноническая транзакция

```bsl
НачатьТранзакцию();
Попытка
    // 1. Блокировка (если нужна)
    Блокировка = Новый БлокировкаДанных;
    ЭлементБлокировки = Блокировка.Добавить("Документ.Название");
    ЭлементБлокировки.УстановитьЗначение("Ссылка", СсылкаНаДокумент);
    Блокировка.Заблокировать();

    // 2. Операции с данными
    ДокументОбъект = СсылкаНаДокумент.ПолучитьОбъект();
    ДокументОбъект.Записать();

    // 3. Фиксация — последняя операция в Попытка
    ЗафиксироватьТранзакцию();
Исключение
    // 4. Откат — первая операция в Исключение
    ОтменитьТранзакцию();

    // 5. Логирование после отката
    ЗаписьЖурналаРегистрации(
        НСтр("ru = 'Операция'"),
        УровеньЖурналаРегистрации.Ошибка,
        Метаданные.Документы.Название,
        СсылкаНаДокумент,
        ПодробноеПредставлениеОшибки(ИнформацияОбОшибке()));

    ВызватьИсключение;
КонецПопытки;
```

## Шаблон 2: исключение с записью в журнал

```bsl
Попытка
    ВыполнитьОперацию();
Исключение
    ИнформацияОбОшибке = ИнформацияОбОшибке();

    ЗаписьЖурналаРегистрации(
        НСтр("ru = 'Имя события'"),
        УровеньЖурналаРегистрации.Ошибка,
        Метаданные.Объекта,
        СсылкаНаОбъект,
        ПодробноеПредставлениеОшибки(ИнформацияОбОшибке));

    ВызватьИсключение;
КонецПопытки;
```

## Шаблон 3: блокировка данных перед чтением

```bsl
НачатьТранзакцию();
Попытка
    Блокировка = Новый БлокировкаДанных;
    ЭлементБлокировки = Блокировка.Добавить("РегистрНакопления.Название");
    ЭлементБлокировки.УстановитьЗначение("Измерение", Значение);
    ЭлементБлокировки.Режим = РежимБлокировкиДанных.Исключительный;
    Блокировка.Заблокировать();

    Запрос = Новый Запрос("ВЫБРАТЬ ...");
    Результат = Запрос.Выполнить();

    // обработка и запись ...

    ЗафиксироватьТранзакцию();
Исключение
    ОтменитьТранзакцию();
    ЗаписьЖурналаРегистрации(...);
    ВызватьИсключение;
КонецПопытки;
```

Подробности порядка операций — в [reference/transactions-canonical.md](reference/transactions-canonical.md) и [reference/locks-managed-and-deadlock.md](reference/locks-managed-and-deadlock.md).
